制作流程

从需求沟通到最终交付，流程清晰，便于修改

沟通需求

确认网页用途、页面数量、文字内容、联系方式、参考风格和是否需要手机端适配。

确认风格

根据需求确定配色、布局、卡片样式、按钮风格和整体视觉方向。

制作初版

完成 HTML、CSS 和 JavaScript 页面开发，生成可直接打开运行的网页文件。

修改交付

根据反馈调整文字、颜色、链接和细节，最终交付 HTML 文件和简单使用说明。
